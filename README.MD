# ToDo Application Backend

### Description : 
- This is a Typescript , Node.js and Rest API backend for Todo-Application.
- Where user can Add tasks, view all tasks , delete tasks and edit tasks options are avalible.

### Features

#### View All Tasks : 
- This feature will display a list of tasks fetches from the database and each task contains : 

    1. Task Name
    2. Description
    3. Status
    4. Priority
    5. Deadline

#### Add Task
- This will allows user to add a new task to database.
- User must provide thise files :
    1. Task Name
    2. Description
    3. Status
    4. Priority
    5. Deadline

####  Edit Task
- User can able to edit or update an existing task by modifing the fields.

#### Delete Task 
- This feature provides user to delete the task permanently from the database.
- It will delete the task by using task id.

### Getting Started

#### Steps : 

1. Clone the project from the remote repositoty.

```bash
https://github.com/shaik-mahammad-rafi01/todo-app-backend.git
```

2. Open the folder.

```bash
cd todo-app-backend
```
3. Install the required dependencies.

```bash
npm install
```
4. Compile the project.

```bash
npm run build
```

5.Run the App.

```bash
npm start
```
