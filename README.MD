# ToDo Application Backend

### Description : 
- This is a Typescript , Node.js and Rest API backend for Todo-Application.
- Where user can Add tasks, view all tasks , delete tasks and edit tasks options are avalible.

### Table of contents 
- [Description](#description)
- [Features](#features)
- [Tech Stack](#tech-stack)
- [Getting Started](#getting-started)
- [Author](#author)

### Features

- View All Tasks 
- Add Task
- Edit Task
- Delete Task

### Tech Stack
- Typescript
- Node.js
- Firebase database
- Express

### Getting Started

#### Steps : 

1. Clone the project from the remote repositoty.

```bash
https://github.com/shaik-mahammad-rafi01/todo-app-backend.git
```

2. Open the folder.

```bash
cd todo-app-backend
```
3. Install the required dependencies.

```bash
npm install
```
4. Compile the project.

```bash
npm run build
```

5.Run the App.

```bash
npm start
```
